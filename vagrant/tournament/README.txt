
There is a Swiss-style tornament Database design in this folder 

This is for project2 on Udacity's fullstack nano-dxcxcxegree

Please make sure that the Database is up by running 

"\i tournament.sql"  

in postgradsql or psql 

This file is in the tournament file where all the other related files are; it 
will create all the nescessary tables and views. 

To run the file please type in the file name preceeded by the command python 

which is as follows:

'python tournament.py' 

Please make sure that this command is run inside the virtual machine in the 
~/vagrant/tournament folder.'

In order to test the database run ' python tournament_test.py ' in the same 
folder stated above.
